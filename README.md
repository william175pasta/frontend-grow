</pre>

  </code>
</pre>
<pre>

  <code>
    var nameList = ['Will', 'Jerry', 'Tom', 'Jack'];
  </code>
</pre>
<pre>
  將Mark排在Jerry後，Bill排在Will前。
  <code>
    var nameList = ['Will', 'Jerry', 'Tom', 'Jack'];
  </code>
</pre>
<pre>
  產生基數列表清單:
  <code>
    var list = [1,3,5,7,...];
  </code>
</pre>
<pre>
  將陣列數值由大到小排列:
  <code>
    var list = [1,3,5,7,9,11];
  </code>
</pre>
